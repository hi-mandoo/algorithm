## 풀이

행의 개수가 10만개가 넘기 때문에 시간초과로 풀 수 없을 것 같아서 DP를 사용해서 풀이했다.  
현재 고른 열과 다른 이전 열들 중에서 최대값을 저장하는 방식으로 배열을 사용했다.

## 코드

```java
class Solution {
    int solution(int[][] land) {
        int[][] dp = new int[land.length][4];

        dp[0] = land[0];

        for (int i = 1; i < land.length; i++) {
            for (int j = 0; j < 4; j++) {
                for (int k = 0; k < 4; k++) {
                    if (j == k) continue;
                    dp[i][j] = Math.max(dp[i][j], dp[i - 1][k] + land[i][j]);
                }
            }
        }

        int max = -1;
        for (int i = 0; i < 4; i++) {
            max = Math.max(max, dp[land.length - 1][i]);
        }

        return max;
    }
}
```
