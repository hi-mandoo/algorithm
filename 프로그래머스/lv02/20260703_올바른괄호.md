## 풀이

스택을 활용하여 풀이하는 문제이다.  
'('이 나왔을 때 스택에 push하고 ')'이 나오면 스택에 있는 괄호를 pop하여 괄호가 올바른지 검사한다.  
만약 pop할 때 스택이 비워져 있거나, 모든 괄호를 검사하고 나서 stack에 괄호가 남아있다면 해당 문자열은 올바르지 않은 괄호이다.

## 코드

```java
import java.util.Stack;

class Solution {
    boolean solution(String s) {
        Stack<Character> stack = new Stack<>();

        for (int i = 0; i < s.length(); i++) {
            if (s.charAt(i) == '(') {
                stack.push(s.charAt(i));
            } else {
                if (stack.isEmpty()) return false;
                stack.pop();
            }
        }

        return stack.isEmpty();
    }
}
```
