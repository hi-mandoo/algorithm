## 의상

- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/42578)

### 풀이

각 종류에 해당하는 의상 수를 알면 정답을 간단한 계산식으로 도출할 수 있는 문제이다.  
종류의 이름에 해당하는 배열의 두 번째 값을 Key로 사용하여 HashMap에 옷들을 저장하여 개수를 세었다.  
전체 데이터를 저장하였다면 의상 배열의 크기에 옷을 고르지 않을 경우의 수 1을 더한 값을 종류 별로 각각 곱한 다음, 최소 한 개 이상의 의상을 착용하기 때문에 옷을 아예 고르지 않는 경우 1을 빼주면 된다.

### 코드

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

public class Solution {
    public int solution(String[][] clothes) {
        Map<String, List<String>> map = new HashMap<>();
        for (int i = 0; i < clothes.length; i++) {
            if (!map.containsKey(clothes[i][1])) {
                map.put(clothes[i][1], new ArrayList<>());
            }
            map.get(clothes[i][1]).add(clothes[i][0]);
        }

        int answer = 1;
        for (Map.Entry<String, List<String>> entry : map.entrySet()) {
            answer *= entry.getValue().size() + 1;
        }
        return answer - 1;
    }
}
```
