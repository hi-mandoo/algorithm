## 대소문자 바꿔서 출력하기

- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181949)

### 문제 풀이

아스키 코드를 이용하여 풀이하면 된다.
대문자와 소문자 사이는 32의 차이가 있으므로 소문자에는 32를 빼주고 대문자에는 더해주면 쉽게 변환할 수 있다.  
Java의 `Character` 클래스에 이미 정의되어있는 `toLowerCase()`나 `toUpperCase()`를 사용하여 더 가독성 있게 코드를 작성할 수 있지만 문자열이 길어질수록 실행시간이 길어진다. 추후 심화된 문제에서 사용하면 시간초과로 문제를 해결하지 못할 수도 있으니 유의하자.

### 코드

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class Solution {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String input = br.readLine();

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < input.length(); i++) {
            if (input.charAt(i) >= 'a' && input.charAt(i) <= 'z') {
                sb.append(Character.toChars(input.charAt(i) - 32));
            } else sb.append(Character.toChars(input.charAt(i) + 32));
        }

        System.out.println(sb);
    }
}
```
