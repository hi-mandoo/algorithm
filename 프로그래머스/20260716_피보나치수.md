## 피보나치 수

- [문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/12945)

## 풀이

문제를 제대로 안읽고 풀어서 int형 오버플로우 발생해서 틀렸다.  
문제에서는 답안을 1234567로 나눠서 제출하도록 되어있다.  
피보나치를 10만까지 계산하면 수가 매우 커지기 때문에 **모듈러 연산**을 사용하여 수를 나눠주면된다.

- 모듈러 연산

```
(a + b) % m == ((a % m) + (b % m)) % m
```

## 코드

```java
class Solution {
    public int solution(int n) {
        int prev = 0;
        int curr = 1;

        for (int i = 2; i <= n; i++) {
            int result = (prev + curr) % 1234567;
            prev = curr;
            curr = result;
        }

        return curr;
    }
}
```
