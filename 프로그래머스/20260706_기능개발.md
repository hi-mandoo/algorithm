## 기능개발

- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/42586)

## 풀이

시간 계산 할 때 타입에 대한 지식이 부족해서 한 번 틀렸다.  
순서대로 배포되어야 하기 때문에 반복문을 돌면서 현재 배포되는 기능을 세다가 현재 배포되는 기능보다 개발이 오래 걸리는 기능이 생기면 그 기능부터 `count` 변수를 1로 초기화해서 다시 세줬다.

## 코드

```java
import java.util.ArrayList;
import java.util.List;

class Solution {
    public int[] solution(int[] progresses, int[] speeds) {
        List<Integer> answer = new ArrayList<>();
        int releaseDay = (int) Math.ceil((100.0 - progresses[0]) / speeds[0]);
        int count = 1;

        for (int i = 1; i < progresses.length; i++) {
            int currentDay = (int) Math.ceil((100.0 - progresses[i]) / speeds[i]);

            if (releaseDay >= currentDay) {
                count++;
            } else {
                answer.add(count);
                releaseDay = currentDay;
                count = 1;
            }
        }

        answer.add(count);

        return answer.stream().mapToInt(Integer::intValue).toArray();
    }
}
```
