## 무인도 여행

- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/154540)

### 풀이

현재 위치의 상하좌우의 식량을 탐색하는 문제이다. **BFS**를 사용해서 각각의 섬을 탐색했다.

### 코드

```java
import java.util.*;

class Solution {

    // 방문 확인 배열
    private boolean[][] visited;
    // 정답 임시 저장 배열(섬의 개수를 정확하게 모르기 때문에 list 사용)
    private List<Integer> answer;

    // 현재 위치의 상하좌우 탐색을 위한 배열
    private int[] toX = {0, 0, -1, 1};
    private int[] toY = {-1, 1, 0, 0};

    public int[] solution(String[] maps) {
        answer = new ArrayList<>();
        visited = new boolean[maps.length][maps[0].length()];

        // 섬의 일부분을 찾아서 그 부분부터 bfs를 한다.
        // 이미 방문했거나, 바다는 탐색하지 않는다.
        for (int i = 0; i < maps.length; i++) {
            for (int j = 0; j < maps[0].length(); j++) {
                if (visited[i][j] || maps[i].charAt(j) == 'X') continue;
                answer.add(bfs(maps, i, j));
            }
        }

        // 섬이 하나도 없는 경우
        if (answer.isEmpty()) answer.add(-1);

        // 오름차순 정렬
        int[] result = answer.stream().mapToInt(Integer::intValue).toArray();
        Arrays.sort(result);

        return result;
    }

    private int bfs(String[] maps, int x, int y) {
        Queue<Node> q = new ArrayDeque<>();
        q.add(new Node(x, y));
        visited[x][y] = true;

        int result = maps[x].charAt(y) - '0';

        while (!q.isEmpty()) {
            Node curr = q.poll();

            for (int i = 0; i < 4; i++) {
                int tx = curr.x + toX[i];
                int ty = curr.y + toY[i];

                // 배열 범위를 넘거나 이미 방문했던 위치, 바다는 탐색하지 않는다.
                if (tx < 0 || tx >= visited.length || ty < 0 || ty >= visited[0].length
                        || maps[tx].charAt(ty) == 'X'
                        || visited[tx][ty]) {
                    continue;
                }

                visited[tx][ty] = true;
                result += maps[tx].charAt(ty) - '0';
                q.add(new Node(tx, ty));
            }
        }

        return result;
    }

}

// 좌표 저장용 클래스
class Node {
    int x;
    int y;

    public Node(int x, int y) {
        this.x = x;
        this.y = y;
    }
}
```
