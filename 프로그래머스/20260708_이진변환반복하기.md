## 이진 변환 반복하기

- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/70129)

### 풀이

단순하게 문제에서 시키는 대로 코딩했다가 런타임 에러가 났다.  
자바에서 제공하는 메서드를 사용해서 쉽게 풀이하려고 했는데, `Integer.parseInt()`는 32비트까지만 변환할 수 있어서 긴 문자열이 입력되었을 때 `NumberFormatException`이 발생했다. 결국 반복문 종료 조건인 1은 숫자 변환 없이 `String` 변환만으로 해결된다. 조건문을 변경해서 해결했다.  
문제의 조건대로만 코딩하면 된다.

### 코드

```java
class Solution {

    public int[] solution(String s) {
        int[] answer = {0, 0};

        while (s.equals("1")) {
            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < s.length(); i++) {
                if (s.charAt(i) == '0') {
                    answer[1]++;
                    continue;
                }
                sb.append(s.charAt(i));
            }

            s = Integer.toBinaryString(sb.length());
            answer[0]++;
        }

        return answer;
    }
}
```
