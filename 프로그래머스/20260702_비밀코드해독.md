## 비밀 코드 해독

- [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/388352)

### 풀이

브루트 포스(Brute Force)로 1부터 n 범위 내의 가능한 모든 비밀 코드를 먼저 찾고, 파라미터로 주어진 해독 결과에 일치하는 비밀 코드 조합의 수를 세어 정답을 구했다.

### 코드

```java
class Solution {
    int n;
    int[][] q;
    int[] ans;

    boolean[] password;
    int answer; // 가능한 조합의 수

    public int solution(int n, int[][] q, int[] ans) {
        // 변수 초기화
        this.n = n;
        this.q = q;
        this.ans = ans;

        answer = 0;
        password = new boolean[n + 1];

        // 1번째 자리 비밀코드 부터, 1부터 시작
        dfs(0, 1);

        return answer;
    }

    private void dfs(int depth, int number) {
        if (depth == 5) { // base case
            for (int i = 0; i < q.length; i++) {
                int count = 0;
                for (int j = 0; j < 5; j++) {
                    if (password[q[i][j]]) count++;
                }

                if (count != ans[i]) return;
            }
            answer++;
            return;
        }

        for (int i = number; i <= n; i++) {
            password[i] = true;
            dfs(depth + 1, i + 1);
            password[i] = false;
        }
    }
}
```
